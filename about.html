O HTML é a estrutura básica da navegação na internet. É a organização e rearranjo dos códigos dessa linguagem de programação que permite a você, usuário, está nesse momento navegando na internet e visitando seus sites e páginas favoritos. A organização desses códigos é o que permite a criação de um hipertexto, que, por sua vez, são os elementos que levam à criação de imagens, textos, vídeos, sons e outros elementos gráficos diversos. Como fazer um HTML com o bloco de notas?

Os instrumentos para a criação de uma página web HTML são tão simples que podem até assustar os desavisados. Após ter estruturado os códigos e tags para seu website, basta fazer o seguinte:

  1. Abrir o bloco de notas do Windows ou qualquer editor equivalente em seu computador;
  2. copiar e colar o código HTML já montado para dentro desse arquivo;
  3. renomear o nome do arquivo para o título da página que você deseja criar;
  4. salvar o arquivo com a extensão “.html” no final;
  5. agora que o arquivo já está salvo no seu computador, basta clicar em cima dele com o botão direito do mouse, selecionar a opção “abrir com” e definir seu navegador web preferido para abrir a página;
  6. prontinho! O site será automaticamente aberto como uma página web no navegador.
